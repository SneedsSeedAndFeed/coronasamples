-- [. About .] --------------------------------------------------------------------------

- Demo on rotating 3D objects and rendering them to screen
- Render modes: Vertices, Wireframe, Flat filled, Light Shaded and Glenz
- Rendering system needs to be improved since larger objects will require more rect objects
- Allows for rotation on all axis, and positioning with x,y,z offsets
- Switch render modes using the buttons
- Minimal optimising
